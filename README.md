# UK-Banking-Performance-Dashboard-2026
An interactive Power BI dashboard analyzing service quality, digital ease, and physical branch trust across major UK banks.

Live Features
Executive Summary Pop-up: An interactive info panel triggered by the "i" icon using Bookmarks and Selection logic.

Service Quality Leaderboard: A clustered bar chart identifying market leaders like Monzo and Starling.

Digital vs. Physical Correlation: A scatter plot analyzing the gap between mobile app satisfaction and branch trust.

Dynamic KPI Cards: Real-time tracking of National Satisfaction (63%), Digital Ease (73%), and Branch Trust (49%).

# Tools Used
Power BI Desktop: Data modeling, DAX measures (Averages), and UI/UX design.

Data Source: Ipsos Personal Banking Service Quality Survey (Feb 2026).

 # How to Use
Filter by Bank: Use the dropdown slicer at the bottom to view specific bank performance.

View Insights: Ctrl + Click the (i) icon in the header to see the Executive Summary.

Exit Summary: Ctrl + Click the yellow arrow to return to the main view.
